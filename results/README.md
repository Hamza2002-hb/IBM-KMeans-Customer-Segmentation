# IBM K-Means Customer Segmentation

## Project Overview

This project demonstrates how K-Means, DBSCAN, and HDBSCAN clustering algorithms can be applied to real-world datasets using Python and Scikit-learn.

The project includes:

- Synthetic Data Clustering
- Customer Segmentation
- DBSCAN Clustering
- HDBSCAN Clustering
- Data Visualization
- Learning Notes
