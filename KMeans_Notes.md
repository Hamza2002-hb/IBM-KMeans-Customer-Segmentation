# K-Means Clustering Notes

## What is Clustering?

Clustering is an Unsupervised Machine Learning technique that groups similar data together without using labeled data.

---

## Types of Clustering

- K-Means
- Hierarchical
- DBSCAN
- HDBSCAN

---

## K-Means

- Uses centroids
- User selects K
- Assigns data to nearest centroid
- Updates centroid repeatedly

---

## Hierarchical Clustering

Agglomerative

Bottom → Top

Divisive

Top → Bottom

Uses Dendrogram

---

## DBSCAN

Uses Density

Important Parameters

- Epsilon
- MinPts

Detects

- Core Points
- Border Points
- Noise

---

## HDBSCAN

Advanced version of DBSCAN

Better for datasets with varying densities

---
